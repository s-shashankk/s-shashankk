## 👋 About Me

I'm a **Software Engineer at Sasken Technologies** (Motorola Mobility India client), where I design and ship intelligent backend systems that blend **enterprise Java** with **cutting-edge AI**.

- 🔭 Currently engineering **Sherpa** — an AI-powered automation platform that autonomously processes device logs, reasons over anomalies via multi-step LLM chains, and auto-files Jira tickets with zero human intervention
- 🤖 Obsessed with **Multi-Agent AI systems** — I build pipelines where Planner, Worker, and Critic agents collaborate asynchronously to complete complex tasks end-to-end
- 🗣️ Built **Brahma**, a conversational BI assistant that translates plain-English questions into live database queries (NL2SQL) — no dashboards, no SQL knowledge required
- 🏋️ When not coding: Gym, Yoga, Chess, and Karate (1st place, State-level Kumite — yes, I'm that guy 🥋)
